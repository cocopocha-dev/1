# ブロックパズル

8×8マスのブロックパズルゲーム。テトリス風のブロックをドラッグ&ドロップで配置し、縦・横一列を揃えて消していくゲームです。

## 遊び方

1. 画面下部に表示される3つのブロックをドラッグしてボード上に配置
2. 3つすべて配置すると新しいブロックが生成される
3. 縦または横一列が揃うとその列が消えてスコア獲得
4. 全ブロックを消すとボーナス+500点
5. ブロックを置ける場所がなくなるとゲームオーバー

## スコア

- 1ライン消去: 10点
- 複数ライン同時消去: ライン数 × 10 + ライン数 × 5（コンボボーナス）
- オールクリアボーナス: +500点
- ベストスコアはローカルストレージに保存

## 技術仕様

- **言語**: HTML / CSS / vanilla JavaScript（フレームワーク不使用）
- **描画**: Canvas API（高解像度Retina対応）
- **対応**: モバイル（タッチ操作）/ デスクトップ（マウス操作）
- **UI言語**: 日本語

## ファイル構成

```
index.html  - エントリポイント
style.css   - スタイルシート
app.js      - ゲームロジック
```

## 起動方法

```bash
# 任意のHTTPサーバーで配信
npx serve .
# または
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。

## モバイルアプリ化について

iOS / Android 両対応のネイティブアプリとしてリリースする場合は以下の選択肢があります：

- **Capacitor**: Web資産をそのままラップしてネイティブアプリ化（推奨）
- **PWA**: Service Workerを追加してホーム画面に追加可能にする
- **React Native / Flutter**: パフォーマンス重視の場合はネイティブ再実装

## 収益化

- 広告表示: Google AdMob（Capacitor プラグイン経由）
- 広告非表示課金: アプリ内購入（IAP）で広告除去オプション
- 対戦モード: Firebase Realtime Database / WebSocket でリアルタイム対戦実装可能
